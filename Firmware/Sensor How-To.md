# Parts List

- Adafruit Feather M0 Adalogger
- Adalogger FeatherWing - RTC + SD Add-on
- Maxbotix Ultrasonic Rangefinder LV-EZ4
- Adafruit FeatherWing OLED 128x32
- MicroSD Memory Card
- 2 header strips
- 4 stacking headers
- 3 Solid-Core Wires, 22 AWG
- CR1220 battery
- 10050mAh 3.7 volt lithium battery
